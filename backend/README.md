## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## To create new modules, controllers, services using nest cli

```bash
$ npm i -g @nestjs/cli
```

```bash
# To create module
$ nest g mo <module-name>

# To create controller
$ nest g co <controller-name>

# To create service
$ nest g s <service-name>
```
